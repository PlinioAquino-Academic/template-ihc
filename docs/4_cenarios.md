# Cenário de Análise/Problema

> **_NOTE:_**: A equipe deve pensar em cenários existentes na atualidade (que causam problemas para os usuários) e que a interface prevista ajudará a resolver o problema. Cenário de Análise/Problema é uma história triste. Não descreve a solução. Descreve somente o problema.

1. Cenário de Análise/Problema: narrativa que possui atores e título.
2. Questões de Refinamento: definir pelo menos 1 pergunta para cada um dos tipos de elementos do método. Devem ser questões que ainda não foram respondidas no cenário de análise/problema (item 1).
3. Refinamento do Cenário Análise/Problema: usando as questões de refinamento (item 2), complemente o texto produzido no item 1, indicando com [] colchetes o local com o novo texto foi incluído.
